App Title

Dokumentimi i Sass: Sass link

Instaloni Node JS: Download

Pasi te instaloni nodejs do keni mundesin qe te perdorni ne terminal npm

Insaloni sass me komanden: npm install -g sass

Per perdoruesit e mac dhe linux perdorni komanden: sudo npm install -g sass

Pasi komanda e mesiperme eshte ekzekutuar me sukses ju mund te krijoni file me extension .scss dhe ti konvertoni ne file .css duke perdorur komanden: sass [emri i file].scss [emri i file].css

Nese doni qe te krijohet file css ne menyr automatike pas cdo ndryshimi, ekzekutoni vetem nje here ne nje terminal unik komanden: sass --watch [emri i file].scss [emri i file].css
